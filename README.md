# Greves Portugal

Platform to view the current strikes in Portugal.

The data is updated daily through [Observador](https://observador.pt) website scraping.

## Tech Stack

- Frontend: React (Vite)
- API: Node.js
- Database: MongoDB
- Scrapper: Puppeteer
- HTML parser: Cheerio
